## Artefatos Gerados

Arquivos gerados automaticamente que nao participam do runtime principal do frontend.

- `openapi-client/`: cliente OpenAPI gerado via `npm run generate-api` ou `npm run generate-api:local`

Esses artefatos ficam fora de `src/` para reduzir acoplamento com o build principal do painel.
